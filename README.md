# Dark Side (Godot RTS Prototype)

Ship-first real-time strategy inspired by the macro goals of Forged Alliance.

## Current Scope (MVP)
- Start as a flagship and explore island sectors
- Harvest Metal + Energy
- Establish island outposts
- Produce drones and basic naval units
- Build missile silos for long-range strikes
- Defeat enemy flagship or hold objective sectors

## Tech
- Engine: Godot 4.x
- Language: GDScript

## Project Layout
- `docs/` design docs and implementation blueprints
- `scenes/` Godot scenes
- `scripts/` gameplay scripts
- `data/` tuning data for units/buildings

## Next Run Steps
1. Open this folder in Godot 4.x.
2. Set `Main.tscn` as startup scene if not auto-detected.
3. Press Play to run the scaffold scene.
